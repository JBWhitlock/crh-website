# CRH Website

Simple Vite + React + Tailwind site for Cornerstone Recovery Holdings, LLC.

## Local path

Recommended local path:

```powershell
C:\Projects\CRH\Website
```

## Setup

```powershell
cd C:\Projects\CRH\Website
npm install
npm run dev
```

Open the Vite URL shown in the terminal, usually `http://127.0.0.1:5173/`.

## Build

```powershell
npm run build
npm run preview
```

## GitHub / Cloudflare Pages

Recommended build settings:

- Framework preset: Vite
- Build command: `npm run build`
- Build output directory: `dist`
- Root directory: `/`
